# Assets 폴더

이 폴더에 원본 아이콘 이미지를 저장하세요.

## 📁 필요한 파일

### icon-original.png
- **위치**: `assets/icon-original.png`
- **크기**: 1024x1024px 이상 권장
- **형식**: PNG (투명 배경)
- **용도**: 모든 사이즈의 앱 아이콘 생성에 사용

## 🚀 사용 방법

1. CITY QUIZ 로고 이미지를 이 폴더에 `icon-original.png`로 저장
2. 다음 명령어 중 하나 실행:
   - `npm run generate-icons` (Node.js)
   - `python scripts/generate-app-icons-python.py` (Python)
   - `.\scripts\generate-app-icons.ps1` (PowerShell)

자세한 내용은 `APP_ICON_GUIDE.md`를 참고하세요.
